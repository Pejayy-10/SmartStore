## üöÄ Overview

**SmartStore** is a powerful, mobile-first Point of Sale (POS) application built with React Native and Expo. Designed for small to medium retail businesses, cafes, and restaurants, it seamlessly combines sales processing with intelligent inventory tracking and recipe management.

Say goodbye to stock discrepancies and manual pricing calculations. SmartStore handles the math so you can focus on your customers.

---

## ‚ú® Key Features

### üõí Modern Point of Sale (POS)

- **Fast Checkout:** Intuitive grid layout for quick product selection.
- **Smart Cart:** Real-time total calculation with tax and discount support.
- **Visual Product Cards:** Beautiful gradient overlays and clear pricing labels.

### üì¶ Intelligent Inventory

- **Ingredient-Level Tracking:** Track stock raw materials (e.g., flour, milk) automatically as you sell finished products.
- **Smart Stock In/Out:** Quick `+`/`-` adjustment buttons and detailed transaction history logs.
- **Low Stock Alerts:** Visual indicators when ingredients fall below custom thresholds.

### üë®‚Äçüç≥ Recipe Management

- **Cost Calculation:** Create recipes from ingredients and automatically calculate the **Total Cost** and **Cost Per Serving**.
- **Smart Pricing:** Set a desired **Markup %** and get an instant **Suggested Selling Price**.
- **Ingredient Preservation:** Edit recipes without losing ingredient connections.

### üìä Analytics & Insights

- **Sales Reports:** Daily, weekly, and monthly sales summaries.
- **Profit Tracking:** Monitor margins with accurate cost-of-goods-sold (COGS) data.

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or newer)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Expo Go](https://expo.dev/client) app on your mobile device (iOS/Android)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/Pejayy-10/SmartStore.git
   cd SmartStore
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npx expo start
   ```

4. **Run on your device**
   - Scan the QR code with your phone's camera (iOS) or Expo Go app (Android).
   - Press `a` for Android Emulator or `i` for iOS Simulator.
